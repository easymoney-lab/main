# 小程序功能说明

## ✨ 新增功能

### 1. 📊 数据统计
- 显示总记录数
- 分别显示"我"和"TA"的记录数量
- 实时更新统计数据

### 2. 🔍 搜索功能
- 支持按地点搜索
- 支持按内容搜索
- 实时过滤显示结果
- 一键清除搜索关键词

### 3. ⚙️ 设置页面
- **自定义用户名**：可以修改"我"和"TA"的显示名称
- **数据统计**：查看详细的记录统计
- **数据管理**：一键清空所有数据
- **关于信息**：显示小程序版本信息

### 4. 🎨 界面优化
- 顶部操作栏：用户切换 + 设置按钮
- 改进的空状态显示
- 更美观的统计卡片
- 优化的搜索框样式

## 📱 页面说明

### 首页（行程列表）
- **用户切换**：顶部切换"我"和"TA"身份
- **设置按钮**：右上角齿轮图标，进入设置页面
- **统计卡片**：显示总记录和双方记录数
- **搜索框**：快速搜索行程记录
- **行程列表**：显示所有行程，点击查看详情

### 添加行程页面
- 选择当前身份
- 选择时间（默认当前时间）
- 输入地点（必填）
- 输入内容描述（可选）

### 详情页面
- 显示完整行程信息
- 可以删除该条记录

### 设置页面
- 个性化设置：修改用户名
- 数据管理：查看统计、清空数据
- 关于信息

## 🚀 使用流程

1. **首次使用**
   - 打开小程序
   - 点击"设置"标签，修改"我"和"TA"的名字
   - 保存设置

2. **添加行程**
   - 点击"汇报"标签
   - 选择身份、时间、地点、内容
   - 提交行程

3. **查看行程**
   - 在首页查看所有行程
   - 使用搜索框快速查找
   - 点击行程卡片查看详情

4. **切换身份**
   - 在首页顶部切换"我"和"TA"
   - 查看不同身份的行程记录

## 💡 使用技巧

- **快速搜索**：输入地点或内容关键词，实时过滤
- **统计数据**：首页顶部查看双方记录数量
- **自定义名称**：在设置页面修改显示名称，更个性化
- **清空数据**：设置页面可以一键清空所有记录（谨慎操作）

## 📝 注意事项

- 数据存储在本地，清除小程序数据会丢失所有记录
- 建议定期备份重要数据
- 搜索功能支持中文和英文
- 用户名最多10个字符
