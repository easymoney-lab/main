# 微信小程序启动指南

## 方式一：使用微信开发者工具（推荐）

### 1. 安装微信开发者工具

如果还没有安装，请访问：
https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

下载并安装微信开发者工具。

### 2. 打开项目

1. 打开微信开发者工具
2. 点击"+"号或"导入项目"
3. 选择项目目录：`/Users/kd/微信小程序`
4. 填写项目信息：
   - **AppID**：可以使用测试号（选择"测试号"）
   - **项目名称**：情侣行程记录
5. 点击"导入"

### 3. 运行项目

导入后，工具会自动编译并运行小程序，你可以在模拟器中看到效果。

## 方式二：使用命令行（如果已配置CLI）

如果你已经配置了微信开发者工具的CLI，可以运行：

```bash
# 打开项目
cli open --project "/Users/kd/微信小程序"
```

## 项目结构确认

确保以下文件存在：
- ✅ `app.js` - 应用入口
- ✅ `app.json` - 应用配置
- ✅ `app.wxss` - 全局样式
- ✅ `pages/index/` - 首页
- ✅ `pages/add/` - 添加页面
- ✅ `pages/detail/` - 详情页
- ✅ `utils/util.js` - 工具函数

## 常见问题

### 1. 图标文件缺失

如果提示图标文件不存在，这是正常的。小程序仍可正常运行，只是底部导航栏可能只显示文字。

你可以：
- 忽略此警告
- 或者添加图标文件到 `images/` 目录

### 2. AppID 问题

- **开发阶段**：使用"测试号"即可
- **正式发布**：需要注册小程序并获取正式 AppID

### 3. 编译错误

如果出现编译错误：
1. 查看控制台的错误信息
2. 检查文件路径是否正确
3. 确认所有必需文件都存在

## 测试功能

运行后可以测试以下功能：

1. **切换身份**：在首页顶部切换"我"和"TA"
2. **添加行程**：点击底部"汇报"标签，填写行程信息
3. **查看列表**：在首页查看所有行程记录
4. **查看详情**：点击行程卡片查看详情
5. **删除记录**：在详情页删除行程

## 调试

打开控制台查看调试日志：
- `[DEBUG]` - 应用级调试信息
- `[UTIL DEBUG]` - 工具函数调试信息

详细调试说明请查看 `DEBUG.md` 文件。
