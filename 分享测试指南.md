# 未发布时分享小程序给他人使用

## 🎯 方法概览

在未正式发布前，有以下几种方式让他人使用：

1. **预览功能**（最简单，适合快速测试）
2. **真机调试**（适合开发调试）
3. **体验版**（最接近正式版，推荐）
4. **上传体验版**（需要上传代码）

## 方法一：预览功能（最简单）

### 适用场景
- 快速分享给朋友测试
- 临时查看效果
- 不需要复杂配置

### 操作步骤

1. **在微信开发者工具中**
   - 点击工具栏的"预览"按钮（或菜单：工具 > 预览）
   - 或使用快捷键（Mac: Cmd+Shift+P）

2. **生成预览二维码**
   - 工具会自动生成一个二维码
   - 二维码会显示在预览窗口中

3. **分享给他人**
   - 让他人用微信扫描二维码
   - 即可在手机上打开小程序

### 特点
- ✅ 简单快速，无需上传
- ✅ 二维码有效期：30分钟
- ✅ 最多支持10人同时预览
- ⚠️ 需要开发者工具保持运行
- ⚠️ 预览二维码会定期更新

### 注意事项
- 预览时，开发者工具必须保持运行
- 如果关闭工具，预览会失效
- 每次预览会生成新的二维码

## 方法二：真机调试

### 适用场景
- 需要调试功能
- 查看控制台日志
- 测试特定功能

### 操作步骤

1. **在微信开发者工具中**
   - 点击工具栏的"真机调试"按钮
   - 或菜单：工具 > 真机调试

2. **选择调试方式**
   - **自动预览**：生成二维码，扫码后自动连接
   - **手动连接**：输入调试器地址

3. **扫码连接**
   - 用微信扫描二维码
   - 手机上会打开调试版本
   - 可以在开发者工具中查看日志

### 特点
- ✅ 可以查看手机上的日志
- ✅ 可以调试代码
- ✅ 支持断点调试
- ⚠️ 需要开发者工具保持运行
- ⚠️ 需要网络连接

## 方法三：体验版（推荐）

### 适用场景
- 正式测试
- 多人使用
- 不需要开发者工具运行
- 最接近正式版体验

### 操作步骤

#### 第一步：上传代码

1. **在微信开发者工具中上传**
   - 点击工具栏"上传"按钮
   - 填写版本号（如：1.0.0）
   - 填写项目备注（如：体验版测试）
   - 点击"上传"

#### 第二步：设置为体验版

1. **登录微信公众平台**
   - 访问：https://mp.weixin.qq.com
   - 使用小程序管理员账号登录

2. **进入版本管理**
   - 左侧菜单：开发 > 版本管理
   - 找到刚上传的版本

3. **选择体验版**
   - 点击"选为体验版"
   - 确认设置

4. **添加体验者**
   - 进入"成员管理" > "体验者"
   - 点击"添加体验者"
   - 输入体验者的微信号
   - 体验者需要在微信中确认

#### 第三步：分享体验版

1. **获取体验版二维码**
   - 在版本管理中，点击"体验版"
   - 下载体验版二维码

2. **分享给体验者**
   - 发送二维码给体验者
   - 体验者扫码即可使用

### 特点
- ✅ 不需要开发者工具运行
- ✅ 可以长期使用
- ✅ 最多支持100个体验者
- ✅ 最接近正式版体验
- ⚠️ 需要上传代码
- ⚠️ 需要添加体验者

### 体验者权限
- 可以正常使用小程序
- 可以测试所有功能
- 数据会正常保存
- 可以长期使用

## 方法四：分享开发版（临时方案）

### 适用场景
- 快速分享给特定人员
- 不需要添加体验者

### 操作步骤

1. **上传代码**（同体验版第一步）

2. **获取开发版二维码**
   - 在版本管理中，找到"开发版本"
   - 点击"预览"或"二维码"
   - 下载二维码

3. **分享二维码**
   - 发送给需要测试的人员
   - 他们扫码后可以临时使用

### 特点
- ✅ 不需要添加体验者
- ✅ 可以快速分享
- ⚠️ 二维码可能有时效性
- ⚠️ 不如体验版稳定

## 🎯 推荐方案

### 快速测试（1-2人）
→ 使用**预览功能**

### 正式测试（多人）
→ 使用**体验版**

### 开发调试
→ 使用**真机调试**

## 📝 详细操作示例

### 体验版完整流程

```bash
# 1. 准备发布版本
./快速部署.sh

# 2. 在微信开发者工具中上传
# - 点击"上传"
# - 版本号：1.0.0
# - 备注：体验版测试

# 3. 在微信公众平台设置
# - 登录 https://mp.weixin.qq.com
# - 开发 > 版本管理 > 选为体验版
# - 成员管理 > 体验者 > 添加体验者

# 4. 分享体验版二维码
# - 下载二维码
# - 发送给体验者
```

## ⚠️ 注意事项

1. **预览功能**
   - 开发者工具必须保持运行
   - 二维码30分钟有效
   - 最多10人同时使用

2. **体验版**
   - 需要上传代码
   - 需要添加体验者
   - 体验者需要确认

3. **数据隔离**
   - 预览版和体验版数据是分开的
   - 正式版数据也是独立的
   - 测试时注意数据不会同步

4. **权限要求**
   - 预览：任何人都可以扫码使用
   - 体验版：需要添加为体验者
   - 开发版：可能需要特定权限

## 💡 最佳实践

1. **开发阶段**：使用预览功能快速测试
2. **测试阶段**：使用体验版让多人测试
3. **发布前**：确保体验版功能正常
4. **发布后**：保留体验版供后续测试

## 🚀 快速开始

### 最快方式（预览）

1. 在微信开发者工具中点击"预览"
2. 让他人扫描二维码
3. 立即可以使用

### 最稳定方式（体验版）

1. 上传代码到微信公众平台
2. 设置为体验版
3. 添加体验者
4. 分享体验版二维码
