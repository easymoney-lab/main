# 组件按需注入说明

## ✅ 已修复

已在 `app.json` 中添加了组件按需注入配置：

```json
"lazyCodeLoading": "requiredComponents"
```

## 📖 什么是组件按需注入？

组件按需注入是微信小程序的性能优化功能，可以：

1. **减少启动时间**：只加载当前页面需要的组件
2. **降低内存占用**：不加载未使用的组件代码
3. **提升性能**：加快小程序启动速度

## 🔧 配置说明

### lazyCodeLoading 配置项

- **"requiredComponents"**：启用组件按需注入（推荐）
- **不配置或空值**：使用传统方式，所有组件都会加载

### 工作原理

启用后，小程序会：
- 分析每个页面使用的组件
- 只加载当前页面需要的组件
- 按需加载其他页面的组件
