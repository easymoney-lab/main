# 问题排查指南

## 如果看到 "Hello World" 页面

这说明小程序可能显示的是默认模板，而不是我们的行程记录小程序。

### 解决方法：

#### 方法1：确认项目路径
1. 在微信开发者工具中，检查左上角显示的项目路径
2. 确认路径是：`/Users/kd/微信小程序`
3. 如果不是，请重新导入项目

#### 方法2：重新编译
1. 点击工具栏的"编译"按钮（或按 Cmd+R）
2. 等待编译完成
3. 查看模拟器是否显示正确的页面

#### 方法3：检查页面配置
1. 打开 `app.json` 文件
2. 确认第一个页面是 `pages/index/index`
3. 确认所有页面路径都正确

#### 方法4：清除缓存重新编译
1. 点击"工具" > "清除缓存" > "清除文件缓存"
2. 点击"编译"重新编译
3. 查看是否正常显示

### 应该看到的正确界面：

✅ **首页应该显示：**
- 顶部：用户切换按钮（"我" / "TA"）和设置按钮（⚙️）
- 中间：统计卡片（如果有数据）
- 搜索框（如果有数据）
- 行程列表或"还没有行程记录~"提示
- 底部：三个标签（行程、汇报、设置）

✅ **导航栏标题应该是：** "我们的行程"（粉色背景）

### 如果还是显示 Hello World：

1. **检查控制台错误**
   - 打开"调试器"标签
   - 查看"Console"是否有红色错误信息
   - 查看"Network"是否有请求失败

2. **检查文件完整性**
   - 确认所有页面文件都存在
   - 确认 `app.js`、`app.json`、`app.wxss` 都存在

3. **重新导入项目**
   - 关闭当前项目
   - 重新导入项目目录
   - 选择正确的 AppID（测试号）

### 快速验证命令：

在终端运行：
```bash
cd "/Users/kd/微信小程序"
ls -la pages/index/
```

应该看到：
- index.js
- index.wxml  
- index.wxss

如果文件缺失，说明项目不完整。
