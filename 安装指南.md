# 微信开发者工具安装指南

## 方式一：自动安装（推荐）

我已经为你创建了自动安装脚本，运行以下命令：

```bash
cd "/Users/kd/微信小程序"
./安装工具.sh
```

脚本会自动：
1. 检查是否已安装
2. 下载最新版本的微信开发者工具
3. 自动安装到应用程序文件夹

## 方式二：手动安装

### 步骤1：下载微信开发者工具

访问官方下载页面：
**https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html**

选择 **macOS** 版本下载（.dmg 文件）

### 步骤2：安装

1. 双击下载的 `.dmg` 文件
2. 将 `微信web开发者工具.app` 拖拽到 `应用程序` 文件夹
3. 等待复制完成

### 步骤3：首次运行

1. 打开 `启动台`，找到 `微信web开发者工具`
2. 首次运行可能需要授权（系统偏好设置 > 安全性与隐私）
3. 使用微信扫码登录

### 步骤4：导入项目

1. 打开微信开发者工具
2. 点击 `+` 号或 `导入项目`
3. 选择项目目录：`/Users/kd/微信小程序`
4. 填写信息：
   - **AppID**：选择 `测试号`（开发阶段）
   - **项目名称**：情侣行程记录
5. 点击 `导入`

## 验证安装

安装成功后，运行以下命令验证：

```bash
# 检查是否安装成功
ls -la "/Applications/微信web开发者工具.app" 2>/dev/null && echo "✅ 安装成功" || echo "❌ 未找到"
```

## 快速启动

安装后，可以使用以下命令快速打开：

```bash
# 打开微信开发者工具
open -a "微信web开发者工具"

# 或者直接打开项目
open -a "微信web开发者工具" --args --project "/Users/kd/微信小程序"
```

## 常见问题

### 1. 无法打开"微信web开发者工具"，因为无法验证开发者

**解决方法：**
1. 打开 `系统偏好设置` > `安全性与隐私`
2. 点击 `仍要打开` 按钮
3. 或者在终端运行：
   ```bash
   sudo xattr -rd com.apple.quarantine "/Applications/微信web开发者工具.app"
   ```

### 2. 下载速度慢

**解决方法：**
- 使用国内镜像或VPN
- 或者手动下载后安装

### 3. 需要更新版本

**解决方法：**
- 打开微信开发者工具
- 点击 `帮助` > `检查更新`
- 或重新下载最新版本

## 系统要求

- **操作系统**：macOS 10.13 或更高版本
- **内存**：建议 4GB 以上
- **磁盘空间**：至少 500MB 可用空间
